  • Configure

    • Add stage

  • New stage

    • Type
      • deploy

  • Add server group

  • Basic Settings

    • Account
      • cluster-admin
    • Namespace
      • test
    • Datail
      • dubbo-demo-service (与项目保持一致)

      • 上一步Jenkins构建的镜像和filebeat镜像
    • Strategy

    • deployment
      • √
    • Strategy
      • RollingUpdate
    • History Limit
      • 7 (相当于保留7个副本方便回滚)

  • Replicas

    • Capacity
      • 1 （副本启动的数量）

  • Volume Sources

    • Volume Source
      • EMPTYDIR
    • Name
      • logm
    • Medium
      • DEFAULT

  • Advanced Settings

    • DNS POlicy

        • blackbox_scheme
      • value
        • tcp
      • key
        • blackbox_port
      • value
        • 20880
      • key
        • prometheus_io_scrape
      • value
        • true
      • key
        • prometheus_io_path
      • value
        • /
      • key
        • prometheus_io_port
      • value
        • 12346
    • Node Selector (手动调度)
    • Tolerations (污点)

  • Contaioner

    • Basic Settings
      • Environment Variables
        • Name (启动jar包的名字)
          • JAR_BALL
        • Value

          • C_OPTS
        • value
          • -Denv=fat -Dapollo.meta=http://config-test.od.com
      • Volume Mounts
        • Source Name
          • logm
        • Mount Path
          • /opt/logs
      • Probes (探针)

          • Readiness Probe

            • 20880
